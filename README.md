## 👀 What workslocal does

workslocal lets you share a site that runs on your own PC with a public link. It is useful when you want to test web apps, webhook calls, or local tools from another device or service.

It includes:
- Local tunnel support
- Catch mode for request checks
- A web inspector for traffic review
- Simple CLI use
- Support for common developer tasks

## 🔧 Set up your first tunnel

After workslocal starts, it connects your local app to a public link.

Typical use looks like this:
- Start your local website on a port such as 3000
- Open workslocal
- Pick the local port
- Copy the public link
- Share that link with your browser, phone, or webhook service

Example:
- Local address: `http://localhost:3000`
- Public link: a workslocal tunnel URL

If you use a webhook tool, paste the tunnel URL into the callback field.

## 🧭 Catch mode

Catch mode helps when you want to inspect incoming requests before they reach your app.

Use it when you want to:
- Test webhook payloads
- See what data a service sends
- Check headers and request paths
- Pause requests for review

This is useful if you work with test forms, bots, or API tools that send data to your local machine.

## 🔎 Web inspector

The web inspector shows request details in a clear view.

You can use it to:
- Review request data
- Check response details
- Find errors in webhook tests
- See how traffic reaches your local app

If a request does not behave as expected, open the inspector and check the full request path, method, and body.

## 🛠️ Troubleshooting

### The tunnel does not start
- Check that your local app is running
- Make sure the port number is correct
- Restart workslocal and try again

### The public link does not load
- Confirm that the tunnel is still open
- Check your internet connection
- Refresh the page
- Make sure the local app still runs on your PC

### The inspector shows no data
- Send a new request to the tunnel
- Check that catch mode is on if needed
- Open the right tunnel session

### Windows blocks the app
- Right-click the file and choose Run as administrator if needed
- Make sure the file finished downloading
- Extract the ZIP folder before запускing the app
